# Prisma — Creative Studio

Landing page for **Prisma**, a worldwide network of visual artists, filmmakers and storytellers.

## Stack

- ⚡ [Vite](https://vitejs.dev/) + [React 18](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- 🎨 [Tailwind CSS](https://tailwindcss.com/)
- 🎬 [Framer Motion](https://www.framer.motion.com/)
- 🔷 [Lucide React](https://lucide.dev/)

## Fonts

- **Almarai** (300, 400, 700, 800) — global default
- **Instrument Serif** (italic) — accent text

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the project.

## Sections

- **Hero** — full-viewport video background with animated heading
- **About** — scroll-linked per-character text reveal
- **Features** — 4-column card grid with staggered entrance animations
